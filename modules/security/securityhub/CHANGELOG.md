# Changelog

## [v2.0] - 2026-07-01

### Changed

- Changed default value of `linking_mode` from `"ALL_REGIONS"` to `"NO_REGIONS"` for safer default behavior.
- Changed default value of `auto_enable_accounts` from `true` to `false`.
- Changed default value of `auto_enable_standards` from `true` to `false`.
- Variable `configuration_type` no longer has a default value and is now required.

### Removed

- Removed the v1.0 release notes file (`releases/v1.0.md`).
